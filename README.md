# jdevhl-core

A Java library to help to developer with usefully utils (parsers, helpers, etc.).

## How to get

It's in maven central.

    <dependency>
        <groupId>io.github.jdevhl</groupId>
        <artifactId>jdevhl-core</artifactId>
        <version>0.0.2</version>
    </dependency>
